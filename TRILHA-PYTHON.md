## 📖 O que você vai aprender

- Fundamentos da linguagem Python — sintaxe, tipos, estruturas de controle e funções
- Programação Orientada a Objetos em Python — classes, herança e encapsulamento
- Estruturas de dados — listas, dicionários, tuplas e sets
- Resolver problemas com algoritmos e lógica avançada usando Python
- Boas práticas de escrita de código Python

## 🛠️ Projetos sugeridos

Ao final da trilha, você deve conseguir construir:

- **Calculadora no terminal** — aplica funções, condicionais e entrada de dados
- **Gerenciador de contatos** — manipulação de listas e dicionários com CRUD no terminal
- **Jogo da forca** — lógica, strings e estruturas de controle
- **Script de automação** — leitura de arquivos, manipulação de dados e geração de relatório
