## 🤔 O que é Frontend?

Frontend é a camada de apresentação de um sistema. É onde o usuário clica, digita, navega e consome informação. Um desenvolvedor frontend é responsável por:

- Estruturar páginas com HTML semântico
- Estilizar interfaces com CSS responsivo
- Adicionar comportamento com JavaScript
- Construir aplicações reativas com frameworks modernos
- Consumir APIs backend e exibir dados dinamicamente
- Garantir boa experiência em diferentes dispositivos e telas

## 📈 Mercado e demanda

- Alta demanda no Brasil e no mundo — toda empresa com produto digital precisa de frontend
- React é o framework **mais requisitado em vagas** atualmente
- Angular tem forte presença em **empresas corporativas e de grande porte**
- Conhecimento de TypeScript é cada vez mais esperado mesmo em vagas júnior
- Profissionais que dominam frontend **e** conseguem se comunicar com o backend se destacam muito

## 🛤️ Stacks disponíveis nesta trilha

### ⚛️ React
Biblioteca criada pelo Facebook — componentes reutilizáveis, estado reativo e ecossistema enorme.
Ideal para SPAs, dashboards, e-commerces e a maioria das vagas frontend do mercado.

### 🅰️ Angular
Framework completo criado pelo Google — estruturado, opinado e com TypeScript no centro.
Ideal para sistemas corporativos, times grandes e projetos que precisam de padrões rígidos.

> Você não precisa aprender os dois ao mesmo tempo. Escolha um, domine-o bem e migre para o outro quando fizer sentido.
