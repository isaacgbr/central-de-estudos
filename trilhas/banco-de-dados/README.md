## 🤔 O que é Banco de Dados?

Banco de dados é o sistema responsável por armazenar, organizar e recuperar informações de forma eficiente e segura. Em qualquer aplicação real — um e-commerce, um sistema de cadastro, um app de tarefas — os dados precisam ser persistidos em algum lugar.

Um desenvolvedor que entende banco de dados é capaz de:

- Modelar a estrutura de dados de um sistema antes de codar
- Escrever consultas eficientes para buscar e manipular informações
- Integrar o banco com a aplicação backend
- Garantir integridade, segurança e performance dos dados
- Entender o impacto das decisões de modelagem no sistema como um todo

---

## 📈 Por que aprender?

- **Pré-requisito real para o backend** — toda API precisa persistir dados em algum lugar
- SQL é uma das habilidades **mais cobradas em processos seletivos** de todas as stacks
- Entender banco de dados te faz um desenvolvedor **muito mais completo e valorizado**
- A base de SQL serve para qualquer banco relacional — MySQL, PostgreSQL, SQLite, Oracle
- Docker facilita o trabalho local sem precisar instalar o banco diretamente na máquina

---

## 🛠️ O que você vai aprender

### SQL
A linguagem padrão para bancos de dados relacionais. Com SQL você cria tabelas, insere dados, faz consultas, filtra, agrupa, ordena e relaciona informações entre tabelas.

### MySQL com Node.js
Como conectar um banco MySQL a uma aplicação Node.js — queries parametrizadas, pool de conexões e integração com Express.

### Docker
Como subir um banco de dados em container localmente — sem instalação, sem conflito de versão, pronto para usar em minutos.
