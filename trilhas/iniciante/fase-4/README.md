Nesta fase você dará seus primeiros passos na linguagem de programação mais utilizada no desenvolvimento web.

Você aprenderá como transformar páginas estáticas em sistemas interativos, entendendo como programas são executados no navegador e como a lógica ganha vida na prática.

---

## 📖 O que você vai aprender

* O que é JavaScript e como ele funciona no navegador
* Como declarar variáveis e armazenar dados
* Como tomar decisões dentro de um programa
* Como repetir ações automaticamente
* Como organizar códigos para resolver problemas reais
