# 01 — Introdução ao JavaScript

Até agora você aprendeu lógica, algoritmos e controle de versão.

Agora começa uma nova etapa:

Você finalmente vai conversar com o computador usando uma linguagem de programação real.

Essa linguagem é o JavaScript.

---

## 🌐 O que é JavaScript?

JavaScript é uma linguagem de programação usada para criar interatividade em páginas da web.

Com ele é possível:

* Detectar cliques em botões
* Validar formulários
* Atualizar conteúdos da tela sem recarregar a página
* Criar animações e efeitos visuais
* Construir sistemas completos que rodam no navegador

Se HTML cria a estrutura e CSS cria o visual, o JavaScript cria o comportamento.

---

## 🧠 Como o JavaScript funciona?

Quando alguém acessa um site:

1️⃣ O navegador carrega o HTML (estrutura)
2️⃣ Aplica o CSS (estilo visual)
3️⃣ Executa o JavaScript (comportamento e lógica)

O JavaScript é interpretado diretamente pelo navegador.

Isso significa que ele é executado imediatamente, sem necessidade de instalação extra.

---

## 💻 Exemplo conceitual

Imagine um botão de login.

Sem JavaScript:
O botão não faz nada.

Com JavaScript:
O sistema pode verificar se o usuário preencheu os campos corretamente e mostrar mensagens na tela.

JavaScript permite que páginas reajam às ações do usuário.

---

## ⚙️ O que é programar com JavaScript?

Programar é escrever instruções que o computador deve seguir.

Essas instruções formam um programa.

Por exemplo, um programa pode:

* Calcular a média de notas de um aluno
* Verificar se um produto está em estoque
* Calcular o valor total de uma compra
* Mostrar mensagens diferentes para cada usuário

JavaScript é a ferramenta que permite criar essas instruções para a web.

---

## 🎯 Por que JavaScript é tão importante?

JavaScript é:

* A linguagem padrão dos navegadores
* Uma das linguagens mais usadas no mundo
* Essencial para desenvolvimento web moderno
* Muito utilizada no mercado de trabalho

Aprender JavaScript abre portas para:

* Frontend (interfaces de usuário)
* Backend (servidores e APIs)
* Aplicativos mobile
* Jogos
* Sistemas web completos
