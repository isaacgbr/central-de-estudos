# 02 — Variáveis e Tipos de Dados

Programas trabalham com informações o tempo todo.

Essas informações podem ser:

* Nome de um usuário
* Idade de uma pessoa
* Preço de um produto
* Status de um pedido
* Resultado de um cálculo

Para guardar esses dados usamos **variáveis**.

---

## 📦 O que é uma variável?

Uma variável é um espaço na memória do computador usado para armazenar um valor.

Você pode imaginar como uma caixa com etiqueta.

A etiqueta é o nome da variável.
O conteúdo da caixa é o valor armazenado.

Exemplo conceitual:

```
nome → "Isaac"
idade → 21
preco → 59.90
```

Cada variável guarda uma informação que pode ser usada depois.

---

## 🧠 Por que variáveis são importantes?

Sem variáveis, programas não conseguiriam:

* Guardar informações do usuário
* Fazer cálculos
* Tomar decisões
* Atualizar dados dinamicamente

Elas são a base de qualquer programa.

---

## 🔢 Tipos de dados

Os valores armazenados podem ter tipos diferentes.

### 📌 Texto (String)

Usado para palavras e frases.

Exemplos:

```
"Maria"
"Login realizado com sucesso"
"Produto esgotado"
```

---

### 📌 Número (Number)

Usado para valores numéricos.

Exemplos:

```
10
3.14
-25
199.90
```

---

### 📌 Verdadeiro ou Falso (Booleano)

Usado para decisões lógicas.

Exemplos:

```
verdadeiro
falso
```

Muito comum para verificar condições como:

* Usuário está logado?
* Produto está disponível?
* Pagamento foi aprovado?

---

## 💻 Exemplo prático (conceitual)

Imagine um sistema de loja online.

```
cliente → "Ana"
idade → 30
totalCompra → 250.00
pagamentoAprovado → verdadeiro
```

O programa usa essas informações para processar a compra corretamente.

---

## 🎯 Boas práticas ao nomear variáveis

Use nomes claros e descritivos:

✅ `nomeCliente`
✅ `totalCompra`
✅ `quantidadeEstoque`

Evite nomes confusos:

❌ `x`
❌ `a1`
❌ `valor2`

Variáveis bem nomeadas facilitam a leitura do código.
