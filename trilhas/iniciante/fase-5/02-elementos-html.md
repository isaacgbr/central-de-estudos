# 02 — Elementos e Organização de Conteúdo

Agora que você já entende que HTML serve para estruturar páginas, vamos conhecer os principais elementos usados para organizar conteúdos.

Esses elementos permitem criar páginas claras, organizadas e fáceis de navegar.

---

## 📝 Títulos e Parágrafos

Títulos são usados para destacar seções importantes da página.

Parágrafos organizam blocos de texto.

Exemplo conceitual:

```id="0t9iua"
<título> Cadastro de Cliente </título>
<parágrafo> Preencha seus dados abaixo </parágrafo>
```

Títulos ajudam o leitor a entender a estrutura do conteúdo.

---

## 📋 Listas

Listas servem para organizar informações em sequência.

### Lista de itens (não ordenada)

Usada quando a ordem não importa.

Exemplo conceitual:

```id="z8s7hf"
• Produto disponível
• Frete grátis
• Entrega rápida
```

### Lista numerada (ordenada)

Usada quando há uma sequência lógica.

Exemplo conceitual:

```id="n1kd82"
1. Escolher produto
2. Adicionar ao carrinho
3. Finalizar compra
```

---

## 🔗 Links

Links conectam páginas diferentes.

Eles permitem navegação dentro de um site ou para páginas externas.

Exemplo conceitual:

```id="ntw3m9"
<link> Ir para página de pagamento </link>
```

Links são essenciais para navegação na internet.

---

## 🖼️ Imagens

Imagens enriquecem o conteúdo visual.

Podem representar produtos, perfis de usuários, banners e ilustrações.

Exemplo conceitual:

```id="9k2mwd"
<imagem> Foto do Produto </imagem>
```

Imagens ajudam a tornar páginas mais atrativas.

---

## 📦 Divisões de Conteúdo

Grandes páginas precisam ser organizadas em blocos.

Esses blocos separam:

* Cabeçalho
* Menu
* Conteúdo principal
* Barra lateral
* Rodapé

Isso mantém a página organizada e facilita estilização depois.

Exemplo conceitual:

```id="30o7sp"
<bloco>
    Conteúdo do produto
</bloco>
```

---

## 🎯 Por que organizar bem o conteúdo?

Boa organização:

* Melhora a leitura
* Facilita navegação
* Ajuda mecanismos de busca
* Torna o site mais profissional
* Melhora acessibilidade

Estrutura bem feita é base para um bom design.

---
