# 01 — Introdução ao HTML

Tudo o que você vê em um site é organizado por uma estrutura invisível.

Essa estrutura é criada usando HTML.

Se compararmos um site com uma casa:

* HTML é a estrutura (paredes, portas, cômodos)
* CSS é a decoração (cores, móveis, estilo)
* JavaScript é o funcionamento (energia, automações)

HTML é o ponto de partida da construção web.

---

## 🌐 O que é HTML?

HTML significa **HyperText Markup Language** (Linguagem de Marcação de Hipertexto).

Ele não é uma linguagem de programação.

É uma linguagem usada para **estruturar conteúdos** em páginas web.

Com HTML você define:

* Títulos
* Parágrafos
* Imagens
* Links
* Listas
* Tabelas
* Formulários
* Divisões de conteúdo

---

## 🧱 Como o HTML funciona?

HTML usa **tags** para marcar cada tipo de conteúdo.

Tags funcionam como rótulos que dizem ao navegador o que cada parte representa.

Exemplo conceitual:

```id="oq94oh"
<título> Página de Perfil </título>
<parágrafo> Informações do usuário </parágrafo>
<imagem> Foto do perfil </imagem>
```

O navegador lê essas marcações e monta a página visualmente.

---

## 📄 Estrutura básica de uma página HTML

Toda página possui uma base estrutural:

```id="y6n0g0"
Documento
 ├── Cabeça (configurações)
 └── Corpo (conteúdo visível)
```

O corpo é onde ficam textos, imagens e demais elementos exibidos ao usuário.

---

## 🧠 HTML organiza o conteúdo

Sem HTML:

Textos e imagens apareceriam soltos e desorganizados.

Com HTML:

O navegador entende:

* O que é título
* O que é parágrafo
* O que é imagem
* O que é menu
* O que é rodapé

Isso permite que a página tenha sentido estrutural.

---

## 🎯 Por que HTML é importante?

HTML é:

* A base de qualquer site
* Essencial para desenvolvimento web
* Necessário para SEO e acessibilidade
* Suportado por todos os navegadores

Todo sistema web começa pela estrutura HTML.
