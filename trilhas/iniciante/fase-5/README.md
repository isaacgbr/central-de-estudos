Nesta fase você aprenderá a construir páginas web completas.

Você entenderá como estruturar conteúdos com HTML e como deixar páginas bonitas e organizadas usando CSS.

Essa é a base de toda interface da internet.

---

## 📖 O que você vai aprender

* Como funciona a estrutura de uma página web
* Como criar textos, imagens, links e listas
* Como organizar conteúdos em blocos
* Como estilizar cores, tamanhos e espaçamentos
* Como deixar páginas visualmente agradáveis
