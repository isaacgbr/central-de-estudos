# 03 — Introdução ao CSS

Você já aprendeu que o HTML cria a estrutura de uma página.

Agora é hora de deixá-la bonita.

É para isso que serve o CSS.

Se HTML é a estrutura de uma casa, o CSS é a pintura, decoração e organização visual.

---

## 🎨 O que é CSS?

CSS significa **Cascading Style Sheets** (Folhas de Estilo em Cascata).

É a linguagem usada para definir a aparência de páginas web.

Com CSS você pode alterar:

* Cores
* Tamanhos
* Espaçamentos
* Fontes
* Alinhamentos
* Bordas
* Sombras
* Layout da página

---

## 🧠 Como o CSS funciona?

O CSS seleciona elementos do HTML e aplica estilos visuais a eles.

Funciona como regras de aparência.

Exemplo conceitual:

```id="q1w2e3"
título {
    cor: azul
    tamanho: grande
}
```

Isso significa:

> “Todo título da página terá cor azul e tamanho grande.”

---

## 🎯 Separação entre estrutura e estilo

Boa prática no desenvolvimento web:

* HTML → Estrutura e conteúdo
* CSS → Aparência visual

Isso permite:

* Organizar melhor o projeto
* Reutilizar estilos
* Facilitar manutenção
* Criar layouts profissionais

---

## 💻 Exemplo prático (conceitual)

Imagine um botão de compra.

Sem CSS:
Botão simples, sem destaque.

Com CSS:

* Cor chamativa
* Texto maior
* Bordas arredondadas
* Efeito ao passar o mouse

CSS melhora a experiência visual do usuário.

---

## 🌈 O que é possível fazer com CSS?

CSS permite criar:

* Sites modernos
* Interfaces responsivas
* Animações suaves
* Layouts organizados
* Experiências agradáveis

Praticamente todo visual bonito da internet usa CSS.
