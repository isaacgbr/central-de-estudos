## 🤔 Por onde começar?

Muita gente começa a programar direto em um framework, sem entender o que está acontecendo por baixo. O resultado é frustração, código copiado sem compreensão e uma base frágil que desmorona quando o assunto fica mais complexo.

Esta trilha existe para evitar isso. Antes de qualquer framework ou tecnologia, você vai aprender a **pensar como um programador** — e isso muda tudo.

---

## 📖 O que você vai aprender

- Pensar de forma lógica e estruturada para resolver problemas
- Escrever algoritmos e entender como o computador executa instruções
- Versionar seu código com Git e colaborar no GitHub
- Escrever seus primeiros programas em JavaScript
- Estruturar páginas com HTML e estilizá-las com CSS

---

## 🗺️ Fases desta trilha

| Fase | Conteúdo |

---

## 🛠️ Projetos sugeridos

Ao final desta trilha você deve conseguir construir:

- **Calculadora no terminal** — aplica lógica, condicionais e funções
- **Lista de tarefas em JavaScript** — manipulação de arrays e objetos
- **Página pessoal** — HTML e CSS do zero, sem framework
- **Repositório no GitHub** — versionado com boas mensagens de commit
