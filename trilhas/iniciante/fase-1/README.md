Antes de aprender qualquer linguagem, você precisa aprender a **pensar como um programador**.  
Lógica de programação é a base de tudo — sem ela, você copia código sem entender o que está fazendo.

---

## 📖 O que você vai aprender

- O que são variáveis e como armazenar informações
- Como tomar decisões no código com condicionais
- Como repetir ações com loops
- Como organizar o código em funções reutilizáveis
