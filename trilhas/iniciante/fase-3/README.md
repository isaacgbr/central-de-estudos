Nesta fase você aprenderá a controlar versões do seu código e a trabalhar em equipe usando as ferramentas mais utilizadas no mercado de tecnologia.

Você entenderá como projetos reais evoluem com segurança, organização e colaboração entre vários desenvolvedores.

---

## 📖 O que você vai aprender
- O que é controle de versão
- Como o Git registra o histórico do projeto
- Como criar versões seguras do seu código
- Como trabalhar em várias funcionalidades ao mesmo tempo
- Como publicar projetos e colaborar usando GitHub
