## 🌐 O que é Java no backend?

Java é uma linguagem compilada, orientada a objetos e fortemente tipada — criada com foco em robustez, segurança e portabilidade. No backend, é usada para construir sistemas que precisam de alta disponibilidade, escalabilidade e confiabilidade.

Com Java no backend você consegue:

- Criar APIs REST robustas e bem estruturadas
- Construir sistemas de grande porte com arquitetura limpa
- Trabalhar com injeção de dependência e padrões de projeto
- Integrar com bancos de dados relacionais e não relacionais
- Desenvolver microsserviços com Spring Boot

## 📈 Mercado e demanda

- Linguagem **mais usada em empresas corporativas, bancos e fintechs no Brasil**
- Alta demanda em **consultorias, empresas de grande porte e concursos públicos**
- Salários competitivos — especialmente com Spring Boot no currículo
- Comunidade madura com décadas de boas práticas consolidadas
- Muito presente em sistemas legados que precisam de manutenção e evolução

---

## ✅ Quando escolher Java para o backend?

- O projeto exige **alta confiabilidade e escalabilidade**
- Você quer trabalhar em **bancos, fintechs ou empresas corporativas**
- O time é grande e precisa de **padrões e convenções claras**
- O sistema precisa de **tipagem estática e segurança de tipos desde o início**
- Você quer uma linguagem com **ecossistema maduro e bem documentado**

## ❌ Quando não é a melhor escolha?

- Projetos pequenos ou MVPs que precisam de **entrega rápida** — a verbosidade do Java aumenta o tempo de desenvolvimento
- Times pequenos sem experiência prévia com a linguagem
- Projetos com **processamento de dados ou machine learning** — Python é mais adequado
- Quando a curva de aprendizado é um bloqueio — Java exige mais tempo para dominar do que JavaScript ou Python

## 🛠️ Framework disponível

### Spring Boot
O framework mais usado no ecossistema Java para desenvolvimento backend. Abstrai grande parte da configuração, segue convenções claras e oferece um ecossistema completo — segurança, banco de dados, autenticação, microsserviços e muito mais.

> Nesta trilha você vai aprender Java puro primeiro — fundamentos, OOP e conceitos avançados — e só então partir para o Spring Boot com base sólida para entender o que o framework faz por baixo.
