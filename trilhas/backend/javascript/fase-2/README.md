Antes de usar qualquer framework, você precisa entender o que o Node.js faz por conta própria.  
Aqui você aprende como um servidor funciona de verdade — sem Express, sem abstração, só o runtime.

---

## 📖 O que você vai aprender

- O que é o Node.js, como funciona seu modelo de execução e por que ele é eficiente
- Como criar um servidor HTTP do zero sem nenhum framework
- Como trabalhar com módulos nativos e o sistema de arquivos
- Quando faz sentido usar Node.js puro e quando um framework é necessário
