Você já sabe como o Node.js funciona por baixo. Agora é hora de usar as ferramentas que o mercado usa.  
Nesta fase você aprende Express e NestJS — entendendo o que cada um faz, como funciona e quando escolher cada um.

---

## 📖 O que você vai aprender

- Criar APIs REST completas com Express — rotas, middlewares e organização de projeto
- Entender o NestJS — sua arquitetura modular e quando ele faz sentido
- Saber com clareza quando usar Express e quando usar NestJS
