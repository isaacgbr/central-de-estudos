## 🎯 Objetivo da Fase

Nesta fase você vai aprender como o **JavaScript funciona fora do navegador**, sendo utilizado para criar sistemas no lado do servidor.

Antes de usar frameworks e ferramentas modernas, é essencial dominar a base da linguagem aplicada ao Backend.

Aqui você entenderá como o JavaScript manipula dados, arquivos e processos no servidor.

---

## 🧠 O que você vai aprender

✅ Diferença entre JavaScript do navegador e do servidor  
✅ Como o JavaScript executa fora do navegador  
✅ Manipulação de variáveis e tipos de dados no Backend  
✅ Estruturas condicionais aplicadas a regras de negócio  
✅ Estruturas de repetição para processamento de dados  
✅ Funções e modularização de código  
✅ Manipulação de arquivos e dados locais
