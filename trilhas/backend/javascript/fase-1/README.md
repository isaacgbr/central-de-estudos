Antes de construir qualquer API, você precisa dominar a linguagem que vai usar.  
Aqui você consolida os fundamentos do JavaScript e aprende os recursos modernos que todo dev backend usa no dia a dia.

---

## 📖 O que você vai aprender

- Revisar e consolidar os fundamentos do JavaScript com foco em backend
- Dominar os recursos modernos do ES6+ — arrow functions, destructuring, spread, módulos e mais
- Entender o que é TypeScript, por que existe e quando vale a pena usar
