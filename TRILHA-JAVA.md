## 📖 O que você vai aprender

- Fundamentos da linguagem Java — sintaxe, tipos, estruturas de controle e funções
- Programação Orientada a Objetos — classes, herança, interfaces, encapsulamento e polimorfismo
- Conceitos avançados — coleções, generics, exceptions, streams e lambdas
- Criar APIs REST completas com Spring Boot
- Estruturar projetos Java seguindo boas práticas do mercado

## 🗺️ Ordem de estudo recomendada

```
1. Java — Parte 1 (fundamentos)
      ↓
2. Java — Parte 2 (OOP)
      ↓
3. Java — Parte 3 (avançado)
      ↓
4. Spring Boot
```

> Não pule etapas — Java tem uma curva de aprendizado estruturada onde cada parte depende da anterior.  
> Spring Boot só faz sentido depois de ter OOP bem consolidado, pois o framework usa esses conceitos intensamente.

## 🛠️ Projetos sugeridos

Ao final da trilha, você deve conseguir construir:

- **Sistema bancário no terminal** — aplica OOP com classes, herança e encapsulamento
- **Gerenciador de estoque** — coleções, generics e manipulação de dados com streams
- **API REST com Spring Boot** — endpoints completos, integração com banco de dados e boas práticas de projeto
