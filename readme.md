## 🗺️ Trilhas disponíveis

### 🟢 [Iniciante](./trilhas/iniciante/README.md)
Para quem está começando do zero. Cobre lógica de programação, algoritmos, Git & GitHub e JavaScript básico — a base que todas as outras trilhas dependem.

### 🔵 [Backend](./trilhas/backend/README.md)
Para quem quer construir APIs e sistemas do lado do servidor. Disponível em três stacks:
- [JavaScript — Node.js](./trilhas/backend/javascript/README.md) — ideal para quem quer entrar rápido no mercado
- [Java — Spring Boot](./trilhas/backend/java/README.md) — ideal para empresas corporativas e fintechs
- [Python — Flask / FastAPI](./trilhas/backend/python/README.md) — ideal para quem tem interesse em dados e IA

### 🟣 [Frontend](./trilhas/frontend/README.md)
Para quem quer construir interfaces modernas. Cobre React, Angular, Vite, Axios e Chart.js.

### 🗄️ [Banco de Dados](./trilhas/banco-de-dados/README.md)
Para quem quer entender como os dados são modelados, armazenados e consultados. Cobre SQL, MySQL com Node.js e Docker.

## 📌 Como funciona

1. **Escolha uma trilha** de acordo com seu objetivo
2. **Siga as fases** — cada trilha é dividida em fases progressivas
3. **Consulte os guias** em `docs/` quando precisar de referência rápida
4. **Use os templates** para iniciar projetos sem boilerplate
