## 📌 Como usar

1. Escolha a trilha que corresponde ao seu objetivo atual
2. Acesse o arquivo da trilha — ele explica o que você vai aprender, os pré-requisitos e a ordem de estudo
3. Consulte os guias de cada tópico como referência rápida durante os estudos
4. Use os templates para iniciar projetos sem configurar do zero
