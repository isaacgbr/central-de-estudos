## 📖 O que você vai aprender

- Criar servidores e APIs REST com Node.js e Express
- Estruturar projetos backend em camadas (routes, controllers, services, models)
- Proteger rotas com autenticação JWT e middlewares
- Entender e configurar CORS para comunicação entre frontend e backend
- Integrar o backend com banco de dados MySQL
- Escrever código backend em PHP como alternativa ao Node.js

## 🗺️ Ordem de estudo recomendada

```
1. Node.js & Express
      ↓
2. MySQL com Node.js
      ↓
3. Autenticação & JWT
      ↓
4. CORS
      ↓
5. PHP (opcional — linguagem alternativa)
```

> Siga essa ordem — Node.js e Express são a base de tudo nessa trilha.  
> MySQL com Node.js vem logo depois para você já praticar com dados reais.  
> PHP é opcional e pode ser estudado de forma independente.

---

## 🛠️ Projetos sugeridos

Ao final da trilha, você deve conseguir construir:

- **API REST de cadastro** — endpoints de criação, leitura, atualização e exclusão com Node.js e Express
- **Sistema de login** — autenticação completa com JWT, rotas protegidas e middleware de verificação
- **API com banco de dados** — integração completa com MySQL, usando queries e boas práticas de segurança
- **API em PHP** — CRUD simples para entender a diferença entre as duas abordagens
