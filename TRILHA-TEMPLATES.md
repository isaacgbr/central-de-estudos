## 📚 Templates disponíveis

### JavaScript
| Template | Descrição |
|----------|-----------|
| [template-crud.js](./templates/js/template-crud.js) | Estrutura base para operações de criação, leitura, atualização e exclusão |
| [template-class.js](./templates/js/template-class.js) | Estrutura de classe com construtor, métodos e encapsulamento |
| [template-async.js](./templates/js/template-async.js) | Funções assíncronas com async/await e tratamento de erros |

### Node.js / Express
| Template | Descrição |
|----------|-----------|
| [server.js](./templates/nodejs-express/server.js) | Servidor Express base com middlewares essenciais configurados |
| [controller.js](./templates/nodejs-express/controller.js) | Controller padrão com métodos para cada operação do CRUD |
| [service.js](./templates/nodejs-express/service.js) | Camada de serviço separando regras de negócio do controller |
| [routes.js](./templates/nodejs-express/routes.js) | Roteamento organizado com separação por recurso |

### Autenticação
| Template | Descrição |
|----------|-----------|
| [middleware.js](./templates/autenticacao/middleware.js) | Middleware de verificação de token JWT nas rotas protegidas |
| [jwt-utils.js](./templates/autenticacao/jwt-utils.js) | Funções utilitárias para geração e validação de tokens JWT |

### Configuração
| Template | Descrição |
|----------|-----------|
| [.gitignore](./templates/git/.gitignore) | Gitignore configurado para projetos Node.js |
| [base.html](./templates/html/base.html) | Estrutura HTML semântica com meta tags e links essenciais |
| [base.css](./templates/css/base.css) | Reset CSS, variáveis de cor e tipografia base |

## 🗺️ Como usar um template

1. Navegue até a pasta do template desejado
2. Copie o arquivo para o seu projeto
3. Renomeie e adapte conforme a necessidade
4. Consulte o guia da tecnologia correspondente caso tenha dúvidas
