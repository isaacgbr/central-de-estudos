## 📖 O que você vai aprender

- Pensar de forma lógica e estruturada para resolver problemas
- Escrever algoritmos e entender estruturas de controle (condições, loops, funções)
- Versionar código com Git e colaborar no GitHub
- Escrever seus primeiros programas em JavaScript
- Estruturar páginas com HTML e estilizá-las com CSS

---

## ✅ Pré-requisitos

Nenhum. Esta é a trilha de entrada.

## 🗺️ Ordem de estudo recomendada

```
1. Lógica de Programação
      ↓
2. Algoritmos
      ↓
3. Git & GitHub
      ↓
4. JavaScript
```

> Siga essa ordem — cada etapa usa o que foi aprendido na anterior.  
> Lógica e algoritmos treinam o raciocínio antes de você escrever código de verdade.

---

## 🛠️ Projetos sugeridos

Ao final da trilha, você deve conseguir construir:

- **Calculadora no terminal** — aplica lógica, condicionais e funções
- **Lista de tarefas em JS** — manipulação de arrays e objetos
- **Página pessoal** — HTML e CSS do zero, sem framework
- **Repositório no GitHub** — versionado com boas mensagens de commit
