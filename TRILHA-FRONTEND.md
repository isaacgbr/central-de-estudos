## 📖 O que você vai aprender

- Construir interfaces reativas com React e gerenciar estado de componentes
- Desenvolver aplicações estruturadas e escaláveis com Angular
- Configurar projetos modernos com Vite como bundler
- Realizar requisições HTTP a APIs externas com Axios
- Criar gráficos e visualizações de dados com Chart.js
- Entender CORS e como lidar com ele no frontend

## 🗺️ Ordem de estudo recomendada

```
1. Vite
      ↓
2. React  ──ou──  Angular
      ↓
3. Axios
      ↓
4. Chart.js (opcional — para projetos com visualização de dados)
      ↓
5. CORS (consulte quando necessário)
```

> Comece pelo Vite — ele será usado para configurar os projetos React e Angular.  
> Escolha React ou Angular de acordo com seu objetivo; não é necessário aprender os dois ao mesmo tempo.  
> Axios e Chart.js são complementos que você usará dentro dos projetos com os frameworks.

---

## 🛠️ Projetos sugeridos

Ao final da trilha, você deve conseguir construir:

- **App de tarefas em React** — componentes, estado, hooks e CRUD consumindo uma API
- **Dashboard em Angular** — listagem de dados, serviços e integração com API REST
- **Painel com gráficos** — visualização de dados com Chart.js integrado ao React ou Angular
- **Consumo de API pública** — requisições com Axios, tratamento de erros e exibição dinâmica dos dados
